Template JSON Builder (Dialog / Foley)

Erzeugt dialog_templates.json oder foley_templates.json im Format [{id,name,url}].
Output: dialog_templates.json
Der Dateiname wird automatisch passend gesetzt.
Wenn gesetzt: URLs werden als basePath + filename erzeugt.
Drag & Drop Audio-Dateien oder einen Ordner hier rein oder nutze die Buttons unten. Unterstützt .mp3 .wav .ogg .m4a .aac .flac
Tipp: Wenn du per Folder-Upload assets/dialog/… reinziehst, bleibt der relative Pfad korrekt.
Dieses JSON entspricht genau dem Template-Bank-Format, das dein Projekt lädt.
Einträge
Du kannst Name/URL in der Tabelle direkt bearbeiten. ID wird aus Name abgeleitet (slug) und eindeutig gemacht.
ID Name URL
Wichtig: Browser können nicht “echte” lokale Pfade schreiben. Für korrekte URLs nutze Folder-Upload oder stelle Base-Pfad passend ein.
